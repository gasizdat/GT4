# Образование отчеств

**Источник:** https://zags.admin-smolensk.ru/poleznaya-informaciya/obrazovanie-otchestv/

## Образование и написание отчеств

Согласно п.4 ст. 18 Федерального закона от 15.11.1997 № 143-ФЗ «Об актах гражданского состояния» отчество ребенка записывается по имени отца, если иное не основано на национальном обычае.

Отчество — это особым образом оформленное имя отца данного человека, входящее в состав его именования и являющееся характерной чертой русской именной системы.

Приводимые правила распространяются лишь на случаи, когда родители хотят дать своим детям отчество по русскому образцу.

## Правила образования отчеств

Отчества от мужских имен (русских и нерусских) в русском языке образуются по следующим правилам:

1. **Твёрдый согласный** (кроме ж, ш, ч, щ, ц) — добавляется **-ович / -овна**
   - Александр + ович/овна
   - Иван + ович/овна
   - Гамзат + ович/овна

2. **Окончание ж, ш, ч, щ, ц** — добавляется **-евич / -евна**
   - Жорж + евич/евна
   - Януш + евич/евна
   - Милич + евич/евна
   - Франц + евич/евна

3. **Неударный гласный а, у, ы** — добавляется **-ович / -овна**, конечные гласные имени отбрасываются
   - Антипа → Антипович/Антиповна
   - Вавила → Вавилович/Вавиловна

   **Исключение:** от русских имён Аникита, Никита, Мина, Савва, Сила, Фока образуются традиционные формы на **-ич / -ична**
   - Никита → Никитич/Никитична
   - Мина → Минич/Минична
   - Савва → Саввич/Саввична

4. **Неударный гласный о** — добавляется **-ович / -овна**, конечный гласный и начальный суффикса сливаются в один звук [о]
   - Василько + ович/овна
   - Михайло + ович/овна
   - Отто + ович/овна

5. **Неударный гласный, предшествуемый ж, ш, ч, щ, ц** — добавляется **-евич / -евна**, гласный отбрасывается
   - Важа → Важевич/Важевна
   - Гоча → Гочевич/Гочевна

6. **Мягкий согласный** (согласный + ь) — добавляется **-евич / -евна**, конечный ь отбрасывается
   - Игорь → Игоревич/Игоревна
   - Цезарь → Цезаревич/Цезаревна
   - Виль → Вилевич/Вилевна
   - Камиль → Камилевич/Камилевна

7. **Неударный гласный е** — добавляется **-евич / -евна**, конечный гласный и начальный суффикса сливаются
   - Аарне → Аарневич/Аарневна
   - Григоре → Григоревич/Григоревна
   - Вилье → Вильевич/Вильевна

8. **Неударный гласный и** — добавляется **-евич / -евна**, конечный гласный сохраняется
   - Вилли → Виллиевич/Виллиевна
   - Илмари → Илмариевич/Илмариевна

9. **Неударное сочетание ий** — добавляется **-евич / -евна**, конечный й отбрасывается, предпоследний и переходит или остаётся

   а) **Переходит в ь** после одного согласного или группы нт
   - Василий → Васильевич/Васильевна
   - Марий → Марьевич/Марьевна
   - Юлий → Юльевич/Юльевна

   б) **Остаётся** после к, х, ц, а также после двух согласных (кроме группы нт)
   - Никий → Никиевич/Никиевна
   - Люций → Люциевич/Люциевна
   - Стахий → Стахиевич/Стахиевна

10. **Старые русские имена на ея и ия** — добавляется **-евич / -евна**, конечное я отбрасывается, и или е сохраняется
    - Менея → Менеевич/Менеевна
    - Захария → Захариевич/Захариевна>
11. **Ударные гласные а, я, е, э, и, ы, ё, о, у, ю** — добавляется **-евич / -евна**, конечный гласный сохраняется
    - Айбу → Айбуевич/Айбуевна
    - Бадма → Бадмаевич/Бадмаевна
    - Бату → Бутуевич/Батуевна
    - Вали → Валиевич/Валиевна
    - Дакко → Даккоевич/Даккоевна
    - Исе → Исеевич/Исеевна

12. **Ударные сочетания ай, яй, ей, эй, ий, ый, ой, уй, юй** — добавляется **-евич / -евна**, конечный й отбрасывается
    - Акбай → Акбаевич/Акбаевна
    - Кий → Киевич/Киевна
    - Матвей → Матвеевич/Матвеевна

13. **Два гласных аа, ау, еу, ээ, ии, уу** — сохраняются, добавляется **-евич / -евна**
    - Бимбии → Бимбииевич/Бимбииевна
    - Бобоо → Бобооевич/Бобооевна
    - Бурбээ → Бурбээевич/Бурбээевна

## Примечания

По настоянию заявителей возможны образования отчеств, которые несколько не соответствуют изложенным правилам, так как с течением времени правила образования отчеств изменяются. Например, отчества от татарских и некоторых других имён имеют тенденцию выравниваться по русским образцам и принимать формы:
- Набиуллович (вместо Набиуллаевич)
- Хамзевич (вместо Хамзяевич)
- Янович (вместо Янисович)
- Мариевич (вместо Мариусович)
- и т. д.

Если у отца именуемого двойное имя, отчество образуется от любого из двух имен по выбору заявителей. Например, если отец Игорь-Эдуард, а ребенка назвали Антон, в его документах пишется:
- либо Антон Игоревич
- либо Антон Эдуардович
- либо Антон Игорь-Эдуардович

как родители сочтут более удобным.
